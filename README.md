# Single-page-application

A clean, responsive, and modern brand landing page built using HTML5, CSS3, and Vanilla JavaScript.

This project demonstrates custom styling, responsive grid layout, smooth scrolling, and reusable utility-style classes inspired by Tailwind design principles.

🚀 Features

✅ Sticky Navigation Bar

✅ Gradient Hero Section with Animation

✅ Responsive Feature Grid (Mobile / Tablet / Desktop)

✅ Call-To-Action Section

✅ Smooth Scroll Functionality

✅ Hover Effects & Micro-Interactions

✅ Fully Responsive Design

✅ Clean & Organized CSS Structure
